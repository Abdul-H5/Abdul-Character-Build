# Abdul-Character-Build
This GitHub Repo is a Character Build assignment for the show Arrested Development. There will 4 characters displayed on a web pages along with a logo and a background image. Two characters will have a lightbox containing information about them. 

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
